⚡ QUICK DEPLOYMENT (Copy-Paste)

1. Push to GitHub:
   git add .
   git commit -m "Ready for deployment"
   git push origin main

2. Generate JWT Secret (copy output):
   node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

3. Deploy Backend (Railway):
   → Go to railway.app
   → New Project → Deploy from GitHub
   → Add PostgreSQL database
   → Set variables (see BACKEND_ENV_TEMPLATE.md)
   → Deploy
   → Copy backend URL

4. Deploy Frontend (Vercel):
   → Go to vercel.com/new
   → Import from GitHub
   → Set root directory: frontend/
   → Add NEXT_PUBLIC_API_URL=http://localhost:4000
   → Deploy
   → Copy frontend URL

5. Connect Backend to Frontend:
   → Railway: Update FRONTEND_ORIGIN to Vercel URL
   → Vercel: Update NEXT_PUBLIC_API_URL to Railway URL
   → Vercel: Redeploy

6. Test:
   → Visit frontend URL
   → Click Login
   → Check DevTools → Network
   → Should see API calls to Railway URL ✓
